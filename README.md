# Smart-Medical-Bracelet-IoT
IoT-based medical bracelet for real-time vitals monitoring (SpO2, BPM, Temp). Features a "Geo-Rescue" logic that sends automated Telegram alerts with live GPS coordinates during health crises to ensure immediate rescue. Built with ESP32 &amp; C++.
